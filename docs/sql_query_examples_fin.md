# SQL examples — PeopleSoft Financials

## GL — balances by account for a period

```sql
SELECT ACCOUNT, SUM(MONETARY_AMOUNT) AS AMT, CURRENCY_CD
FROM PS_LEDGER
WHERE BUSINESS_UNIT = :bu
  AND LEDGER = :ledger
  AND FISCAL_YEAR = :yr
  AND ACCOUNTING_PERIOD = :per
GROUP BY ACCOUNT, CURRENCY_CD
ORDER BY ACCOUNT;
```

## AP — vouchers for a vendor (recent)

```sql
SELECT VOUCHER_ID, INVOICE_ID, INVOICE_DT, GROSS_AMT, POST_STATUS_AP
FROM PS_VCHR_HDR
WHERE BUSINESS_UNIT = :bu
  AND VENDOR_ID = :vndr
  AND INVOICE_DT >= ADD_MONTHS(TRUNC(SYSDATE), -3)
ORDER BY INVOICE_DT DESC;
```

## Vendor name on voucher

```sql
SELECT H.VOUCHER_ID, H.VENDOR_ID, V.NAME1
FROM PS_VCHR_HDR H
JOIN PS_VENDOR V ON V.VENDOR_ID = H.VENDOR_ID
  AND V.SETID = H.VENDOR_SETID
  AND V.EFFDT = (SELECT MAX(V2.EFFDT) FROM PS_VENDOR V2
                 WHERE V2.SETID = V.SETID AND V2.VENDOR_ID = V.VENDOR_ID
                 AND V2.EFFDT <= SYSDATE)
WHERE H.BUSINESS_UNIT = :bu AND H.VOUCHER_ID = :vid;
```

## Discover columns on a record

```sql
SELECT FIELDNAME, FIELDNUM FROM PSRECFIELD WHERE RECNAME = 'VCHR_HDR' ORDER BY FIELDNUM;
```
